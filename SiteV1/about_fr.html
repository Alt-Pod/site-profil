Introduction

Ce site, à l'exception des couleurs, a entièrement été conçu par moi-même.

Il est :

  • Entièrement statique
  • Référencé via schema.org et opengraphprotocol.org
  • Universellement responsive (smartphone, tablette, ordinateur)
  • Génial

Il a entièrement été développé depuis Notepad++ et sublime Text.

Dans les détails

Mon objectif a été de faire un site très simple avec une mise en forme légèrement poussée (surtout vis à vis du multi display). Tout a été fait pour réduire le temps de chargement du site au niveau des images : il y en a le moins possible, sans redimensionnement par le navigateur.

Le site n'est cependant pas optimisé et fait preuve de redondance au niveau du code (surtout css). La relecture à été sommaire et les méthodes utilisée ont été très simple, sans reflexion particulière au niveau d'un modèle très structuré.

La plupart des formes que vous voyez sont entièrement construite en css. Exemples présents sur ce site :

width:0px;

height:0px;

border-top:6px solid transparent;

border-bottom:6px solid transparent;

border-right:6px solid transparent;

border-left:6px solid #94AAB9;
-webkit-transform:skew(-30deg);

-moz-transform:skew(-30deg);

-o-transform:skew(-30deg);

tranform:skew(-30deg);

Le multi display se fait en css essentiellement grâce à :

@media only screen and (max-width:la largeur maximale)

Dont voici quelques exemples :

@media only screen and (max-width:747px){

  .content_header{

    width:59%

  }

  .arrow_4{

    display:block;

    border-top:6px solid #94AAB9;

    border-left:6px solid transparent;

    margin-left:5px;

    margin-bottom:-10px;

  }

  .socialnetwork{

    margin-top:380px;

  }

}
@media only screen and (max-width:688px){

  .content_header h1{

    margin-left:5px;

  }

  .content_header h2{

    font-size:20px;

  }

  .main_menu ul{

    margin-left:-1%;

  }

}

J'ai travaillé principalement ici en terme d'horizontalité (et avec des pixels), ce qui est peu commun. Il y a néanmoins certains éléments réagissant de manières verticale (comme par exemple les deux block de bas de page dans Accueil).
